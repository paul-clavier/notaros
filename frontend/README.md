# joule-muse

# Setup

## joule-spin

Login to GitHub packages to be able to run `npm install`

```sh
npm login --scope=@nw-tech --auth-type=legacy --registry=https://npm.pkg.github.com
```

## node version

Use the same version of node, specified in `.nvmrc`

```sh
nvm use
```

## install

```sh
npm install
```
